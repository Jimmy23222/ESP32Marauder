# Save
Save your current list of SSIDs or Access Points. If you create a list of SSIDs for beacons or you have scanned access points, you can save these items to your SD card to be used later.

## Menu Path
`WiFi`>`General`>`Save/Load Files`

## CLI
This function is available for use via the [Marauder CLI](cli). The following documentation describes command usage.

### Usage
`save -a/-s`

#### Arguments
| Argument | Required/Optional | Description |
| -------- | ----------------- | ----------- |
| `-s` | Optional | Save list of SSIDs |
| `-a` | Optional | Save list of Access Points |
